# Pydantic Enum

Some utils for working with Enums on a Pydantic Model.

## Features

- Automatic conversion of `IntEnum` to `str`
- Supports Python versions: 3.10, 3.12
- Supports Pydantic versions: v1.X, v2.X

## Run tests

```sh
poetry run tox
```
